## 📌 Project Overview

This project presents an end-to-end Business Intelligence solution developed using **Microsoft Power BI** to analyze Uber ride booking data. The dashboard transforms raw transactional data into meaningful business insights that help monitor operational performance, customer behavior, revenue trends, fleet utilization, and geographical demand.

The project demonstrates practical applications of **data cleaning, data modeling, DAX calculations, KPI design, interactive visualization, and business storytelling**. It is designed to simulate a real-world business analytics scenario where decision-makers require fast, interactive, and reliable reports for strategic planning.

---

## 🎯 Business Problem

Ride-sharing platforms generate thousands of bookings every day. As the volume of operational data increases, extracting meaningful insights manually becomes difficult. Business stakeholders require a centralized dashboard to answer questions such as:

- How many rides are completed every month?
- Which vehicle category generates the highest revenue?
- What are the major reasons for ride cancellations?
- Which locations experience the highest demand?
- Which payment methods are most frequently used?
- How can operational efficiency be improved?

This dashboard addresses these business challenges by providing an interactive analytics solution that enables users to monitor performance and make data-driven decisions.

# 📂 Dataset Overview

The dataset contains ride-level transactional information representing Uber bookings.

### Major Attributes

| Column | Description |
|---------|-------------|
| Booking ID | Unique ride identifier |
| Booking Status | Completed, Cancelled, etc. |
| Booking Value | Revenue generated |
| Vehicle Type | Ride category |
| Pickup Location | Starting location |
| Drop Location | Destination |
| Ride Distance | Total distance travelled |
| Customer Rating | Customer feedback |
| Driver Rating | Driver feedback |
| Payment Method | Cash, UPI, Wallet, Card |
| Booking Date | Date of ride |
| Time Slot | Ride timing |

The data has been organized to support monthly and quarterly business analysis.
